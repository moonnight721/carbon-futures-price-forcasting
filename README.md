## 專案動機與目標
- **問題背景**：碳價受能源、股票與市場情緒等多因素影響，具有跨市場與非線性特性。
- **研究目標**：以 **多特徵深度學習模型** 預測 **歐盟碳排放權（EUA）期貨開盤價**，並支援 **t+1 至 t+24** 的前視步長。
- **專案特色**
  1. 跨市場特徵整合（歐洲 / 美國 / 能源面）。
  2. 多模型可切換（RNN / LSTM / GRU 與 CNN-序列混合）。
  3. **GRA 灰關聯分析** 做特徵初篩，提升可解釋性。
  4. 統一輸出（csv/圖），便於快速比較與論文撰寫。

## 資料與特徵
**目標變數**
- `EUA_Open`：EUA 期貨開盤價（預測標的）

**主要特徵（可自由組合）**
- `EUA_*`：歐洲碳權期貨  
- `VIX_*`：美國恐慌指數  
- `DJIA_*`：美國工業指數  
- `DJCIEN_*`：美國能源商品指數  
- `NG_*`：美國天然氣  
- `STOXX50_*`：歐洲前 50 公司指數  
- `Oil_*`：布蘭特原油  
- `GAS_*`：荷蘭 TTF 天然氣期貨  
- **ALL**：EUA + 全部額外特徵

> ⏱ **時序對齊建議**：考量歐美時差，常以「前一交易日（lag1）」的美股/能源變數對齊當日 `EUA_Open`。
